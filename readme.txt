WWM Registration Form Widget for trading platform registration. Simple and flexible. 

== Description ==
WWM Registration Form Widget can manage multiple trading platform registration form on your own WordPress site.
The form supports ajax submitting, multilingual,customize themecolor and so on.

= Docs & Support =

You can find [docs](http://support.worldwidemarkets.com/wp-content/plugins/WWM-Registration-Form-Widget/readme.html).
If you were unable to find the answer to your question in any of the documentation, please post a topic for it or send us an email.

== Installation ==

1.Download 'WWM Registration Form'. Unzip the folder and upload the entire folder to '/wp-content/plugins/' directory.

2.Active the plugin through the 'Plugins' menu in WordPress.

You will find the widget available in you WordPress admin panel widget page.

== Frequently Asked Questions == 

= Where to set up basic values? =

You can go to settings option or edit within the shortcode/Widget.

= Can I user IB Hash and Affiliate ID at the same time? =

No.You are only allowed to use IB Hash and Agent ID, OR Affiliate ID and Click ID.
