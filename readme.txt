WWM Registration Form Widget for trader account registrations using AlphaTrader or MetaTrader 4. Simple and flexible. 

== Description ==
The WWM Registration Form Widget can manage and process AlphaTrader/MetaTrader 4 client registration forms using WorldWideMarkets API - directly from your own WordPress site. Sidebar widget(s) and/or standalone shortcode(s) are available for easy installation.

This plugin seamlessly integrates with WorldWideMarkets IB Explorer and Affiliate program for Forex trader account referrals. 

The form supports Ajax submission, multilingual functionality, lead source tracking/email notifications, complete customization of theme color and more.

== Installation ==

1.Download 'WWM Registration Form'. Unzip the folder and upload the entire folder to '/wp-content/plugins/' directory.

2.Active the plugin through the 'Plugins' menu in WordPress.

You will find the widget available in you WordPress admin panel widget page.

== Frequently Asked Questions == 

= Where to set up basic values? =

You can go to settings option or edit within the shortcode/Widget.

= Can I user IB Hash and Affiliate ID at the same time? =

No.You are only allowed to use IB Hash and Agent ID, OR Affiliate ID and Click ID.
